# Lession 1

## Data Types
There are some common types
1. Number
- integer
    - int: 4 bytes
    - long long: 8 bytes

- float
    - float: 4 bytes
    - double: 8 bytes
2. char
- 1 byte

3. Bool
- 1 bytes
- 2 values: true/false

## Variables
```cpp
Data_Type variable_name;
```
- condition:
    - do not start with a digit
    - do not include space
    - must not be any reserved word or keyword
